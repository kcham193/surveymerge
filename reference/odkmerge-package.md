# odkmerge: Merge and Flatten ODK and KoboToolbox Repeat-Group Exports

Provides tools to read, detect, enrich, and merge repeat-group sheets
produced by Open Data Kit (ODK) and KoboToolbox XLSForm exports.
Automates the joining of parent and child sheets using standard ODK
index columns, supports multi-level and nested repeat groups, and
produces clean flat datasets ready for analysis.
