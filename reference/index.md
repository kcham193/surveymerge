# Package index

## Main entry point

The one-call wrapper for most users.

- [`odk_merge()`](https://kcham193.github.io/odkmerge/reference/odk_merge.md)
  : Merge an ODK or KoboToolbox export into a flat dataset

## Step-by-step workflow

Use these when you need finer control.

- [`read_odk_export()`](https://kcham193.github.io/odkmerge/reference/read_odk_export.md)
  : Read an ODK or KoboToolbox Excel export
- [`detect_structure()`](https://kcham193.github.io/odkmerge/reference/detect_structure.md)
  : Detect the parent-child sheet structure of an ODK export
- [`print(`*`<odk_structure>`*`)`](https://kcham193.github.io/odkmerge/reference/print.odk_structure.md)
  : Print an odk_structure object
- [`build_master()`](https://kcham193.github.io/odkmerge/reference/build_master.md)
  : Build a flat master dataset from ODK repeat-group sheets
- [`enrich_repeat()`](https://kcham193.github.io/odkmerge/reference/enrich_repeat.md)
  : Enrich a repeat sheet with columns from its parent sheet
